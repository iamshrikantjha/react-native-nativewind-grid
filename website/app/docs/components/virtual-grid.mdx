# Virtual Grid (Performance)

Rendering massive grids efficiently.

## Usage

Use `<VirtualGrid />` for lists with hundreds or thousands of items. It behaves like `FlatList`.

```tsx
import { VirtualGrid } from 'react-native-nativewind-grid';

<VirtualGrid
  className="grid-cols-3 gap-2"
  data={data}
  renderItem={({ item }) => <View className="h-40 bg-gray-200" />}
/>
```
