# API Reference

## `<Grid />`

| Prop | Type | Description |
| :--- | :--- | :--- |
| `className` | `string` | Tailwind classes. |
| `masonry` | `boolean` | Enable masonry mode. |

## `<VirtualGrid />`

| Prop | Type | Description |
| :--- | :--- | :--- |
| `data` | `Array` | Data source. |
| `renderItem` | `Function` | Item renderer. |
